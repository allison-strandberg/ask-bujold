# Ask the Author: Lois McMaster Bujold — searchable archive

A static site for browsing Lois McMaster Bujold's 1500+ [Goodreads Ask the Author](https://www.goodreads.com/author/16094.Lois_McMaster_Bujold/questions) answers (1,500+ of them).

Unofficial fan project. All answers are Bujold's own words, sourced from her public Goodreads Q&A, with a link back to Goodreads on every question.

## Layout

- `index.html`, `assets/` — the site. Plain HTML/JS/CSS; search runs client-side with [MiniSearch](https://github.com/lucaong/minisearch). No build step.
- `data/questions.json` — the full scraped corpus (questions, answers, likes, comments). Goodreads spoiler shrouds are resolved to their hidden text at scrape time, with a `spoiler` boolean per question; the site blurs those with click-to-reveal (the "Hide spoilers" checkbox, on by default).
- `data/tags.json` — tags per question ID. Hand-editable; the scraper never touches it.
- `data/taglist.json` — tag display order/grouping for the site (generated by `autotag.py`).
- `scraper/scrape.py` — the scraper. Full scrape every run: all list pages (refreshes text, likes, comment counts for the whole corpus), then comment threads only where the comment count changed. Refuses to write a dataset smaller than the existing one.
- `scraper/autotag.py` — first-pass keyword tagging. Additive: only tags question IDs not already in `tags.json`, so hand-curation survives re-runs.
- `scraper/scrape_ages.py` — one-time capture of each answer's relative age ("8 years ago") from question pages → `data/answer_ages.json`. Never needs re-running; new questions are dated by when the archive first sees them.
- `scraper/infer_dates.py` — estimates when each answer was posted (`data/dates.json`, regenerated every run) by combining first-comment timestamps, the one-time page ages, and `first_seen`, then enforcing newest-first monotonicity. Cards show it as "answered ~May 2026"; hover explains the source.

## Running locally

```sh
python3 -m venv .venv && .venv/bin/pip install -r scraper/requirements.txt
.venv/bin/python scraper/scrape.py     # ~15 min first run, minutes after
.venv/bin/python scraper/autotag.py
python3 -m http.server 8000            # then open http://localhost:8000
```

Scraper flags: `--max-pages N` (test run), `--list-only` (skip comments), `--delay S` (default 1.2s between requests — keep it polite), `--use-cache` (re-parse from `.cache/` without hitting Goodreads).

## Updates

`.github/workflows/scrape.yml` re-scrapes weekly (Mondays), auto-tags anything new, and commits if the data changed. With GitHub Pages set to deploy from the branch root, a data commit redeploys the site automatically. The job fails loudly rather than committing a shrunken/partial dataset.

## Design notes

Vorkosigan livery (brown and silver) as the base theme; the five gods' seasonal colors — Daughter's spring blue, Mother's summer green, Son's autumn red-orange, Father's winter grey, and the Bastard's white — as the header stripe and tag accents.
